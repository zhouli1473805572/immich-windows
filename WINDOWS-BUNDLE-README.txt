Immich Windows Native
=====================

Open README.md in this folder and use the language tabs:

  中文启动说明
  English Start Guide

Quick start:

  1. Extract the whole archive.
  2. Run build\ImmichManager.exe.
  3. Click Start All.
  4. Open http://127.0.0.1:2283.
